Hi! This is the blog about the Pokemon Serenity and Strife project!
This project is a scratch project that has started a few years before. We create an entire Pokemon game on Scratch, and we add our own pokemons.
We recently rebooted the project, so our actual team is too small to be listed here.

You would like to join us?

Get yourself a new Scratch account if you don't have any and go on this Scratch Topic to register yourself!
You'll first need to create a project to show us your skills, then apply by posting on the thread, with the link to this project, your name and which job you'd like (Map maker, Spriter, Artist, Programmer...)
